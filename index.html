Why you'll love Eventool

Eventool is an event managment system which computerize the needs of the production. Producers can watch details about events, like how many clients arrived, how much did they pay and which promoter invited them.

In addition, users can watch and add comments and details on every client, like for which events he arrived, for which he didn’t arrive and who are his friends.

Cashiers will also use the system - can mark arrival of client and add client if doesn’t exist. With the system, producers can learn from past events - which themes are better, which promoters are more successful, what’s the price the clients are willing to pay etc…

The application uses machine learning algorithm in order to suggest promoters which clients are relevant for current event.
